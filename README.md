# Restaurant Landing Page

Responsive single-page restaurant landing page.

## Features
- Semantic HTML5
- Responsive layout (desktop / tablet / mobile)
- CSS Grid and Flexbox
- Anchor-based navigation with smooth scrolling
- Reusable UI components
- Accessible forms and navigation
- SVG icons

## Sections
- Header & navigation
- Hero
- About (mosaic layout)
- Reservation & working hours
- Menu preview
- Newsletter
- Footer

## Tech Stack
- HTML5
- CSS3
- Webpack
- TypeScript (project-ready structure)

## Project Structure
src/
 ├─ assets/
 │  ├─ images/
 │  └─ icons/
 ├─ fonts/
 ├─ styles/
 │  ├─ variables.css
 │  └─ style.css
 ├─ index.html
 └─ index.ts

## Getting Started

Install dependencies:
```npm install```

Run the project locally:
```npm run start```

Build for production:
```npm run build```
